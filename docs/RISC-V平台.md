### 选择RISC-V的原因

+ 简单
+ 可定制
+ 具有新特性

X86太复杂，太多需要向前兼容的设计，ARM启动过程缺少标准。并且以上两者都是成熟的商业用品，不能修改定制。

### 关于RISC-V 64

RISC-V 64是一个开源的指令集。

### SBI

SBI的全程是Supervisor Binary Interface，

### OpenSBI

OpSBI是一个开源的SBI，

### RISC-V的特权

RISC-V一共有3种特权级：

+  U Mode (User / Application 模式)
+ S Mode (Supervisor 模式)
+ M Mode (Machine 模式)
